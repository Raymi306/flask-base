# Flask-Base
Inspired by the flask projects I worked on in the past, with a bit of tidying up

## venv + pip for setting up dependencies

`python3.9 -m venv venv`

`source venv/bin/activate`

`pip install -r requirements.txt`

## how to run

### app

`source venv/bin/activate`

`python -m app`

### tests

`source venv/bin/activate`

`python -m unittest`
